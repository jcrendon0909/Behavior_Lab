# BehaviorLab 🚀

AI Agents + Governance + Cybersecurity

## Setup

### 1. Frontend

Deploy en Cloudflare Pages desde /frontend

### 2. Worker

```
npm install -g wrangler
wrangler login
wrangler publish
```

### 3. Variables

* OPENAI_API_KEY
* MONGO_API_URL

### 4. R2

Usar para videos/assets y referenciar en HTML

## Arquitectura

Frontend → Worker → AI → MongoDB
